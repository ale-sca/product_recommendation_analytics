# 📊 Product Recommendation Analytics System

A Streamlit app that evaluates how product recommendations impact **customer retention** and **Customer Lifetime Value (CLV)**.

---

## Who is this for?

* Product teams running growth experiments
* RevOps teams analyzing customer behavior
* Data analysts working on e-commerce metrics

---

## What does it do?

* Simulates an **e-commerce product analytics stack**
* Runs experiments on recommendation strategies
* Measures impact on:

  * Retention
  * Trends in the number of repeated purchases/customers.
  * Average basket size per cohort.
  
* Uses the Olist public dataset for realistic scenarios
